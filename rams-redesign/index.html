The Vault

Python-powered personal wiki workflow that turns Obsidian captures into a maintained Zettelkasten-style knowledge graph. An LLM-assisted pipeline ingests raw notes, updates durable synthesis pages, creates atomic markdown entries, and preserves links so the wiki compounds instead of becoming a static archive.

02

Canvas to Things

Python automation that bridges the Canvas LMS with Things 3 using GitHub Actions and SMTP. A stateful polling system with redundant queuing bypasses the Things Cloud daily email limit and keeps task synchronization reliable across multiple courses.

Repo
03

Touch Sensitive Electric Piano

Realtime MIDI instrument built with Arduino-based capacitive sensors feeding Python and C++ apps. FluidSynth rendering, custom serial messaging, and cross-platform MIDI I/O let keystrokes stream to synthesized audio with reliable latency.

Repo
04

Curtain Call Alarm Clock

SwiftUI alarm app that pairs with an Arduino Nano curtain motor over BLE. Asynchronous device discovery and a redundant signal queue open the curtains when the alarm fires while lowering idle power draw with a low-energy HC-08 handshake.
